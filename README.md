# 🏭 CyberCORE: Smart Factory Dashboard v6.0

Bu proje, yüksek teknolojili bir fabrikadaki robotik üretim hatlarını anlık olarak izlemek ve yönetmek için geliştirilmiş, **Cyberpunk** estetiğine sahip bir yönetim panelidir.

## 🚀 Özellikler
- **Gerçek Zamanlı Veri Takibi:** Üretim bantlarındaki 20+ robot ünitesinin (Robot-X1, X2 vb.) durumunu anlık izleme.
- **Dinamik Grafikler:** Chart.js entegrasyonu ile operasyonel verimlilik ve hata oranlarının görselleştirilmesi.
- **Akıllı Durum Yönetimi:** Robotların "Operasyonel", "Hatalı" veya "Bakımda" durumlarına göre dinamik renk değişimleri.
- **Otomatik Veri Simülasyonu:** Arka planda çalışan Python scriptleri ile gerçekçi fabrika verisi üretimi.

## 🛠️ Teknolojiler
- **Backend:** Python & Django
- **Frontend:** HTML5, CSS3 (Neon UI Design), JavaScript
- **Veri Görselleştirme:** Chart.js
- **Veritabanı:** SQLite3

## 📦 Kurulum
1. Depoyu klonlayın: `git clone https://github.com/keremokn1/CyberCORE-SmartFactory.git`
2. Sanal ortamı oluşturun ve aktif edin: `.\venv\Scripts\activate`
3. Gereksinimleri yükleyin ve sunucuyu başlatın: `python manage.py runserver`
4. Simülasyonu başlatın: `python simulasyon.py`
